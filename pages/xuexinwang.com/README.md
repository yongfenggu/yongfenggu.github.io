# 学信网模拟站 - 配置说明

## 数据流

```
config.js  →  login.html 验证账号密码
           →  localStorage 存储个人信息
           →  各页面读取 localStorage 渲染
```

每次点击登录按钮都会重新加载 config.js（带时间戳防缓存），不会使用旧数据。

## 字体编码说明

学信网使用 **Chsi 私有字体**对文字进行编码（Unicode 私有区 U+100000+），
源码中看到的方框/乱码字符在浏览器中通过 Chsi 字体渲染为正常中文。

获取编码文本的方法：
1. 在真实学信网页面上，用浏览器开发者工具选中目标文字
2. 复制该元素的 textContent（不是肉眼看到的文字，是底层编码字符）
3. 粘贴到 config.js 对应字段


## 添加新账号模板

将以下模板复制到 `config.js` 的 `accounts` 对象中：

```js
    // ↓ 替换为登录用户名
    username: {
      password: "123456",

      // === 以下字段粘贴从学信网复制的编码文本 ===
      姓名: "",
      性别: "",
      出生日期: "",
      民族: "",
      证件号码: "",

      学校名称: "",
      专业: "",
      层次: "",
      学习形式: "",
      学制: "",
      学历类别: "",
      分院: "",
      系所: "",
      班级: "",
      学号: "",

      入学日期: "",
      毕业日期: "",
      学籍状态: "",
      毕结业: "",
      证书编号: "",
      校长姓名: "",

      // === 图片：放到 src/ 目录下 ===
      录取照片: "./src/xxx_lqpic.jpg",
      学历照片: "",
    },
```

## 图片

- 录取照片、学历照片放在 `src/` 目录下
- 文件名建议用 `{用户名}_lqpic.jpg`、`{用户名}_xlpic.jpg` 区分
- 学历照片如果没有，留空字符串 `""`，页面会自动隐藏
